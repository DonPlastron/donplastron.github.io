## Pagina toevoegen
1. Voeg een icoontje toe in het grid: <br>
  - ga naar code.js <br>
  - in de data variabele, voeg een object toe in de juiste array (hobby/films) (zie voorbeeldjes)

<br>
2. Maak de pagina

  - Kopieer "pagetemplate.html" naar juiste folder

  - Verander de naam naar de naam die je invulde als titel in de vorige stap
  - Zet het pad naar de standaard afbeelding die je wilt ipv "pad/naar/afbeelding.jpg" (gwn die div wegoen als je geen foto wilt daar)
  - Vul de inhoud aan in plaats van "Hier komt de inhoud in html opmaak"

<br>
3. Uploaden naar github

  - Ga naar vertakking icoontje op vscode
  - Vul een Message in met wat je hebt toegevoegd
  - Klik Commit. Bij de pop-up "There are no staged changes to commit", klik "Yes".
  - Klik Sync Changes
