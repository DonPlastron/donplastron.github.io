## Pagina toevoegen
1. Voeg een icoontje toe in het grid:
a. ga naar code.js
  b. in de data variabele, voeg een object toe in de juiste array (hobby/films) (zie voorbeeldjes)
2. Maak de pagina
  a. Kopieer "pagetemplate.html" naar juiste folder
  b. Verander de naam naar de naam die je invulde als titel in de vorige stap
  c. Zet een titel ipv "Titel"
  d. Zet het pad naar de standaard afbeelding die je wilt ipv "pad/naar/afbeelding.jpg" (gwn die div wegoen als je geen foto wilt daar)
  e. Vul de inhoud aan in plaats van "Hier komt de inhoud in html opmaak"
